# productivity-app

A mobile-first, AI-native **unified work feed**: one daily, prioritized "what needs me today" across the tools where professionals work (GitHub, Linear, Slack, Gmail). Each item is classified by action type (Reply / Review / Approve / Decide / FYI), ranked by priority, and actionable from the feed itself. Not a to-do list, not a social feed: a triage surface for cross-tool attention.
